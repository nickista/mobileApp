## Claude Ai를 활용한 컨텍스트 메뉴 예제
백그라운드를 꾹 누르고 있으면 컨텍스트 메뉴로 다섯 개의 색이 나타난다.
색을 누르면 백그라운드 컬러가 변경된다.
